# Project Information: Visual Prosthesis Device

## Project Overview
**Visual Prosthesis Device** is an advanced biomedical wearable designed to restore vision through electrical stimulation of the optic nerve. This open-source research project combines neural interface engineering, embedded systems, real-time signal processing, and full-stack web development.
